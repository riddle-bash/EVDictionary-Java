# EVDictionary-Java
English-Vietnamese Dictionary with Online Google Translate

Các thành viên trong nhóm UET K64-K2:
- Hoàng Hải Nam
- Đinh Ngọc Sơn
- Nguyễn Tiến Nghĩa

Phiên bản sử dụng: JDK 17

Framework: SceneBuilder, JavaFX

Technology: MySQL, JDBC, FreeTTS, REST
